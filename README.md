# Hybrid POS System 

A modern, high-performance **Restaurant Management System** built with a hybrid architecture.

## 🛠️ Tech Stack
- **Go (Golang):** Scalable Backend & API management.
- **C:** Low-level logic for high-performance stock calculations.
- **Python:** Modern and user-friendly GUI.

## 🏗️ Architecture
This project focuses on **System Design** by connecting different languages through network protocols and native bindings.
